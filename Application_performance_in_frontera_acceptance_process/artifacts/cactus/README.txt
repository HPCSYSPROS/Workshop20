Cactus
======

For a detailed description of the benchmark problem, see here:
https://bluewaters.ncsa.illinois.edu/spp-benchmarks

Input
=====

The input can be found unmodified on the SPP benchmark website.  It is too large
to include here.

Code modifications
==================

No direct code modifications, but configuration options can be found in:

code/stampede2-victor.cfg

and list of enabled "thorns" of the Cactus framework can be found in:

code/ThornList

Third-party library dependencies
================================

Cactus has the following third-party library dependencies:

1. hdf5 1.8.16
2. Intel 18's MKL
